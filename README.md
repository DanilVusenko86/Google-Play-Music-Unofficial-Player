# Project preparation

Copy that repo using
``` Bash
git clone https://github.com/DanilVusenko86/Google-Play-Music-Unofficial-Player.git
```

Then install modules using *npm*

# Debugging and Testing

For Debugging use
``` Bash
npm start
```
For packaging use
``` Bash
npm run make
```
